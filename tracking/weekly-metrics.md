---

## How to pull numbers from Instagram

1. **Reels:** Profile → Insights → Content → select Reel → see Saves, Shares, Reach
2. **Stories:** Insights → Stories → see Link clicks, Replies per Story
3. **Profile visits:** Insights → Overview → Profile visits → filter by Reels
4. **DMs:** Count manually — search DMs for book names each Sunday
5. **Orders:** Your own order log (spreadsheet or notebook)

---

## Red flags (act immediately)

| Signal | Action |
|--------|--------|
| Reels get views but zero DMs | Stronger CTA on Stories, not Reels |
| DMs but no orders | Simplify payment flow, respond faster |
| Same book asked but out of stock | Restock or suggest alternative in template |
| Low saves on all Reels | Try "Teen kitabein ek mood" list format |
| Story link clicks dropping | Refresh link sticker placement, test WhatsApp direct link |
