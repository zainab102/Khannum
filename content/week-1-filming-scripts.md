# Week 1 — Filming Scripts & Shot Lists

Film all 5 Reels in one session (60–90 min). Same setup: window light, cream/beige background, wood table.

**Before filming:** Charge phone, clean background, prepare 3 books (Namal, Peer-e-Kamil, Raja Gidh), packing materials (tissue, note card, tape).

---

## Reel 1: Brand Intro — "Who We Are"

**Format:** Voiceover + B-roll  
**Length:** 35–45 sec  
**Pillar:** Curator's Pick / Brand identity

### Shot list

| # | Shot | Duration | Notes |
|---|------|----------|-------|
| 1 | Slow pan across 5–8 books on table | 5 sec | Natural light, no hands |
| 2 | Hand picks up one book, opens to first page | 4 sec | Gentle movement |
| 3 | Packing: tissue wrap close-up | 4 sec | From packing ritual |
| 4 | Handwritten note being placed | 3 sec | |
| 5 | Sealed package, still on cream background | 3 sec | |
| 6 | Book cover hold — straight to camera | 4 sec | End card |

### Voiceover script (Urdu — record quiet, warm tone)

```
KHANUM ek curated bookstore hai.

Hum har kitab nahi rakhtay —
sirf woh jo hum genuinely believe karte hain
ke kisi ki shelf par deserve karti hai.

Har order haath se pack hota hai.
Ek delivery nahi — ek experience.

Agar aap ko reading se gehra rishta hai,
to aap sahi jagah par hain.
```

### Text overlays (CapCut)

1. `KHANUM` (opening, 2 sec)
2. `Curated Urdu books` (mid)
3. `DM ya link in bio se order karein` (end, 3 sec)

### Caption

```
Hum books bulk mein nahi bechtay — carefully chuntay hain.

Har order haath se pack hota hai, ek khayal ke saath.

Agar aap ko Urdu kitabon se gehra rishta hai, KHANUM aap ke liye hai.

#KHANUM #KHANUMBooks #UrduBooks #PakistaniBookstore #UrduAdab #BookstagramPakistan #CuratedBooks
```

## Batch Filming Order (efficient)

1. Set up table once (all Reels same session)
2. Film all packing shots first (Reel 2)
3. Film B-roll pan across books (Reel 1)
4. Film Namal, Peer-e-Kamil, Raja Gidh reader match shots (Reels 3–5)
5. Record all voiceovers in one quiet take (edit in CapCut)
6. Edit Reel 2 first (easiest — no voiceover), post same day for momentum
