## Reusable Story Templates (use every week)

### Template A — Stock Update

```
[Book cover photo]

[BOOK NAME]
Stock mein ✦

[One line — who it's for]

DM "[BOOK NAME]" to order
```
Sticker: Link to WhatsApp / bio

---

### Template B — Packing Clip

```
[5–10 sec packing video]

Aaj ka order pack ho raha hai.
Aap ka bhi aise hi jayega.
```
Sticker: Message

---

### Template C — Poll (2x/week)

Options ideas (rotate):
- "Aap kya padh rahe hain?" — Urdu novel / English / Spiritual / Poetry
- "Gift ke liye kitab?" — Fiction / Spiritual / Poetry / Surprise me
- "[Book] padhi hai?" — Haan / Nahi / Dobara padhungi

---

### Template D — Q&A (1x/week)

Sticker text: `Konsi kitab recommend karun?`

Reply publicly on Story (not DM) with book name + one honest line. End with: "DM karein agar order karna ho."

---

### Template E — Social Proof (2x/week)

```
[Blurred DM screenshot]

Shukriya [first name only] —
aap ka order safely pahunch gaya.

Agar share karna chahein to tag karein.
```

---

### Template F — Low Stock (only when true)

```
[Book cover]

[BOOK NAME]
Sirf [X] copies baqi hain.

DM jaldi karein — honestly scarce hai.
```

---

## Story Sales Flow (every sales Story)

1. Show book (2 sec)
2. One honest line about who it's for
3. "DM book name ya link in bio se order karein"
4. Never: countdown timers, fake urgency, "BUY NOW"
