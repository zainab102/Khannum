# KHANUM — DM Reply Templates

Copy-paste and fill in `[BRACKETS]`. Keep tone warm, never pushy.

---

## 1. First inquiry (customer asks about a book)

```
Assalam o Alaikum!

[BOOK NAME] abhi stock mein hai.
Price: Rs. [PRICE]
Delivery: [X–X] din, Pakistan-wide

Order confirm karne ke liye apna naam, phone aur address bhej dein —
hum haath se pack karke bhejenge.

Koi aur sawal ho to pooch lein.
```

---

## 2. Price only asked

```
[BOOK NAME] — Rs. [PRICE] (+ delivery if applicable)

Agar order karna ho to naam aur address bhej dein.
```

---

## 3. Multiple books inquiry

```
Assalam o Alaikum!

Aap ki list:
• [BOOK 1] — Rs. [PRICE]
• [BOOK 2] — Rs. [PRICE]
• [BOOK 3] — Rs. [PRICE]

Subtotal: Rs. [TOTAL]
Delivery: Rs. [DELIVERY] (ya free over Rs. [X])

Confirm karne ke liye naam, phone, address bhej dein.
```

---

## 4. Order confirmed — payment request

```
Shukriya! Aap ka order confirm:

[BOOK NAME] × [QTY]
Total: Rs. [TOTAL] (delivery included)

Payment:
JazzCash / EasyPaisa / Bank transfer
[ACCOUNT DETAILS]

Payment screenshot bhej dein — phir pack karna shuru karenge.
```

---

## 5. Payment received — packing started

```
Payment receive ho gayi — shukriya!

Aap ka order pack ho raha hai.
[BOOK NAME] haath se wrap karke [X] din mein dispatch karenge.

Tracking [agar hai] yahan bhejenge.
```

---

## 6. Order dispatched

```
Aap ka order dispatch ho gaya!

Courier: [NAME]
Tracking: [NUMBER]

Jab receive ho to batayein — humein khushi hogi.
Agar share karna chahein to @khanumm.official tag karein (koi pressure nahi).
```

---

## 7. Book out of stock

```
Shukriya ke aap ne [BOOK NAME] poochi.

Abhi stock khatam hai — jese hi aaye gi hum Stories par update karenge.
Agar chahein to wait list par rakh sakte hain, ya koi similar suggest karun?
```

---

## 8. Recommendation request

```
Batayein thora — aap ko kya pasand hai?

• Spiritual / imaan
• Love story with depth
• Long fiction (Namal type)
• Classic adab
• Poetry

Ek line mein bata dein — main honestly recommend karungi.
```

**After they reply:**

```
Aap ke liye [BOOK NAME] achi rahe gi.

[One honest line — who it's for]
Price: Rs. [PRICE]

Agar dil kare to order karein — warna koi baat nahi.
```

---

## 9. Hesitant customer (not sure)

```
Koi jaldi nahi. Padhna ek khushi hai — jab dil kare tab order karein.

Agar koi specific sawal ho book ke baare mein to pooch lein,
main honestly bataungi ke yeh aap ke liye hai ya nahi.
```

---

## 10. Thank you after delivery

```
Assalam o Alaikum! Order receive ho gaya?

Umeed hai sab theek ho. Agar koi masla ho to batayein.

Agar aap comfortable hon to thank-you message share kar sakte hain
(naam blur kar denge) — lekin sirf agar dil kare.
```

---

## 11. Honest scarcity (only when true)

```
[BOOK NAME] — sirf [X] copies baqi hain.

Hum ne yeh title kam rakha hai. Jab khatam ho jaye to
dubara wait ho sakti hai. Bata dein agar reserve karna ho.
```

---

## 12. WhatsApp redirect (from Instagram)

```
Instagram DM slow ho sakta hai —
WhatsApp par message karein for faster reply:

[WHATSAPP LINK]

Wahan book ka naam likh dein.
```

---

## Words to NEVER use

- "Limited time offer!!"
- "Don't miss out"
- "Last chance" (unless literally last copy)
- "Swipe up NOW"
- "Hurry!!"
- Fake countdown urgency

## Words that work for KHANUM

- "Agar dil kare"
- "Honestly"
- "Haath se pack"
- "Carefully chuni"
- "Aap ke liye lagti hai to"
- "Koi pressure nahi"
